# ChurchCMS Android Mobile App

The ChurchCMS Android app brings your church to your pocket. As a member, you can access key church information, events, and community features from your smartphone.

## What You Can Do

- 📅 **View Events** - See upcoming services, events, and activities
- 👥 **Browse Directory** - Find and connect with fellow members
- 🙏 **Prayer Requests** - Submit and view community prayers
- 📰 **Announcements** - Stay updated with church news
- 📹 **Watch Sermons** - Access recorded sermons and media
- 🔔 **Get Notifications** - Receive alerts for events and updates

## First Time Setup

### Step 1: Launch the App

After installation, open ChurchCMS mobile app.

### Step 2: Enter Server Details

1. You'll see the **"Server Connection"** screen
2. Enter your **ChurchCMS instance URL**:
   - Example: `https://churchcms.myparish.com`
   - Example: `http://192.168.1.100:8000` (local network)
3. Tap **"Connect"**
4. App verifies connection to the API

### Step 3: Log In

1. Enter your **email address**
2. Enter your **password**
3. Select your **church** (if you have multiple)
4. Tap **"Sign In"**

**Note:** Credentials are the same as your web application login

### Step 4: Grant Permissions

Android will request permissions for:
- **Camera** - For profile photo uploads
- **Contacts** - For member directory syncing
- **Location** - For event location mapping
- **Notifications** - For push alerts
- **Storage** - For media caching

Grant permissions to enable full functionality.

### Step 5: Start Using

Once logged in, explore:
- **Dashboard** - Overview of church info and announcements
- **Members** - Church directory and member profiles
- **Events** - Upcoming services and events
- **Prayer Requests** - Submit and view prayer needs
- **Messages** - Communicate with other members

## Configuration

### Change Server

1. Tap **Menu** (☰) or your **Profile**
2. Select **"Settings"**
3. Tap **"Change Server"**
4. Enter new ChurchCMS URL
5. Tap **"Connect"**
6. Log in again

### Update Profile

1. Tap **Profile** icon
2. Tap **"Edit Profile"**
3. Update:
   - Profile photo
   - Phone number
   - Address
   - Bio
4. Tap **"Save"**

### Notification Settings

1. Go to **Settings**
2. Tap **"Notifications"**
3. Enable/disable:
   - Events
   - Prayer requests
   - Messages
   - Announcements
   - Surveys

### Offline Mode

The app automatically caches:
- Member directory
- Event information
- Sermon listings
- Downloaded content

Access cached data without internet connection.

## Troubleshooting

### Can't Connect to Server

**Problem:** "Unable to connect to server" error

**Solutions:**
1. Verify server URL is correct
2. Check internet connection
3. Ensure server is running and accessible
4. Try entering just the domain without `/api`
5. If using self-signed SSL: Configure trust in settings

### Login Fails

**Problem:** "Invalid credentials" error

**Solutions:**
1. Verify email and password are correct
2. Check CAPS LOCK is off
3. Reset password on web application if forgotten
4. Try logging in on web app first to confirm access
5. Contact your church administrator

### Notifications Not Working

**Problem:** Not receiving push notifications

**Solutions:**
1. Check notification permission is granted
2. Verify notifications are enabled in app settings
3. Check phone's notification settings for the app
4. Ensure battery saver mode is not blocking notifications
5. Reinstall app if issue persists

### Slow Performance

**Problem:** App is slow or freezing

**Solutions:**
1. Clear app cache: **Settings → Apps → ChurchCMS → Storage → Clear Cache**
2. Close unnecessary apps running in background
3. Check available storage space (you need at least 100 MB free)
4. Update app to latest version
5. Reinstall app if issues continue

### Storage Space Issues

**Problem:** "Storage full" or "Cannot cache media"

**Solutions:**
1. Go to **Settings → Storage**
2. View cached media size
3. Clear old cache: **Settings → Storage → Clear Cache**
4. Delete cached sermons/photos you no longer need
5. Free up phone storage by deleting other apps

### Connection to Old Server

**Problem:** App keeps trying to connect to old server

**Solutions:**
1. Log out completely
2. Go to **Settings → Change Server**
3. Clear all stored data
4. Reinstall app (back up any local data first)
5. Set up again with new server details

## FAQ

**Q: Will the app work without internet?**
A: Partially. Cached data (directory, events, sermons) are available offline. New content requires connection.

**Q: Can I use one account on multiple devices?**
A: Yes, log in on multiple Android devices with same credentials.

**Q: Are my photos and data secure?**
A: Yes, data is encrypted in transit and at rest. Your privacy settings control visibility.

**Q: Can church staff monitor my activity?**
A: Only permissions you grant. Prayer requests, attendance, events are tracked as configured by your church.

**Q: What happens if I uninstall the app?**
A: All cached data is deleted. You can reinstall anytime and log back in.

**Q: Does it work with WhatsApp/SMS?**
A: App handles in-app messaging. For SMS integration, contact your administrator.
