# ChurchCMS

**ChurchCMS** is a comprehensive, open-source web application licensed under the MIT License. It's designed to help churches manage members, events, finances, and communications efficiently.

## Architecture

ChurchCMS consists of multiple integrated components:

- **Web Application + API** - Built with Laravel (PHP), providing a robust backend and REST API
- **Android Mobile App** - Native Android application for on-the-go access
