---
title: Add Members to a Group
description: Guide to adding members to a group.
---


<Step>
  <StepItem title="Click Add Members">
    Click **Add Members**.

    ![Add Members Button](/modules/groups/add-members-button.png)
  </StepItem>
  <StepItem title="Search and select members">
    Search and select members from the dropdown list.

    ![Search Members](/modules/groups/search-members.png)
  </StepItem>
  <StepItem title="Assign a Member Role">
    Assign a **Member Role**.
  </StepItem>
  <StepItem title="Set permissions">
    Set permissions by selecting authorization options:
    - **Read**
    - **Create**
    - **Update**

    ![Set Permissions](/modules/groups/set-permissions.png)
  </StepItem>
  <StepItem title="Members listed below">
    Added members will be listed below.
  </StepItem>
  <StepItem title="Submit">
    Click **Submit**.
  </StepItem>
</Step>
