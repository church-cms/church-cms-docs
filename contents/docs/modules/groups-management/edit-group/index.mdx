# Edit Group

<Step>
  <StepItem title="Click Edit Group">
    Click **Edit Group**.

    ![Edit Group Button](/modules/groups/edit-group-button.png)
  </StepItem>
  <StepItem title="Update the required details">
    Update required details such as:
    - **Cover Image**
    - **Members** (add/remove)
    - **Roles and permissions**

    ![Edit Group Form](/modules/groups/edit-group-form.png)
  </StepItem>
  <StepItem title="Submit the form">
    Click **Submit**.
  </StepItem>
</Step>
