# Viewing Group Details

<Step>
  <StepItem title="Click the Group Name">
    Click on the required **Group Name**.

    ![Groups List1](/modules/groups/groups-list1.png)
  </StepItem>
  <StepItem title="Group details page opens">
    The group details page will open.
  </StepItem>
  <StepItem title="Manage the group">
    You can manage the group using the available options:
    - **Add Members**
    - **Send Message**
    - **Message List**
    - **Edit Group**
    - **Delete Group**
  </StepItem>
</Step>
