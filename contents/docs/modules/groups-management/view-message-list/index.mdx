---
title: View Message List
description: Guide to viewing the message list in a group.
---


<Step>
  <StepItem title="Click Message List">
    Click **Message List**.

    ![Message List Button](/modules/groups/message-list-button.png)
  </StepItem>
  <StepItem title="View message details">
    View details such as:
    - **Message Sent To**
    - **Mode** (Email / Notification / SMS)
    - **Sent On**
    - **Status**
    - **Action**

    ![Message List1](/modules/groups/message-list1.png)
      ![Message List2](/modules/groups/message-list2.png)
  </StepItem>
</Step>

<Note type="note">
  Messages can be filtered based on **Mode**.
</Note>
