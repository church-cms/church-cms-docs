# Send Message to Group Members

<Step>
  <StepItem title="Click Send Message">
    Click **Send Message**.

    ![Send Message Button](/modules/groups/send-message-button.png)
  </StepItem>
  <StepItem title="Enter the required details">
    Enter the required details:
    - **Subject**
    - **Message**
    - **Attachments** (if any)

    ![Send Message Form](/modules/groups/send-message-form.png)
  </StepItem>
  <StepItem title="Send the message">
    Click **Send**.
  </StepItem>
</Step>

<Note type="note">
  Messages can be sent via:
  - **Email**
  - **Notification**
  - **SMS**
</Note>
