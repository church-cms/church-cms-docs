---
title: How to Edit a Gallery
description: Guide to editing an existing gallery album.
---

<Step>
  <StepItem title="Navigate to Gallery">
    Navigate to the **Gallery** section.
  </StepItem>
  <StepItem title="Locate the album">
    Locate the required album.
  </StepItem>
   <StepItem title="Name of the desired folder">
    Click the name of the desired folder option on the left bottom of the album image and add the images to be listed in the folder.
  </StepItem>
     <StepItem title="Select Photos">
   Select the photos to be added and click submit. 

  ![Gallery Section Folder](/modules/gallery/gallery-section-folder.png)
  ![Gallery Section upload](/modules/gallery/gallery-section-upload.png)
  </StepItem>
         <StepItem title="Photos added to the folder">
  The photos will be added in the specified folder in the gallery 

    ![Photos Count](/modules/gallery/photos-count.png)
  </StepItem>
  <StepItem title="Click Edit">
    Click the **Edit** option on the right side of the album image and change the cover image if required.

    ![Gallery Section](/modules/gallery/gallery-section.png)

  </StepItem>
  <StepItem title="Update the details">
    Update the necessary details:
    - **Gallery Name**
    - **Change Cover Image**

    ![Edit Album Form](/modules/gallery/edit-album-form.png)

  </StepItem>
  <StepItem title="Submit the form">
    Click **Submit** to save the changes.
  </StepItem>
</Step>

## Outcome

- The gallery details will be updated successfully.
