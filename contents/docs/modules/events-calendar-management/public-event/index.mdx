# Creating a Public Event

<Step>
  <StepItem title="Select Public Event">
    Click **Create Public Event** from the dropdown menu.
  </StepItem>
  <StepItem title="Fill in the details">
    Fill in all the required event details (same as private event).
  </StepItem>
  <StepItem title="Submit the form">
    Click **Submit**.
  </StepItem>
</Step>

## Outcome

- The public event will be created successfully.
- The event will be visible in the calendar.
- Notifications will be sent to members through the member application.

<Note type="note" title="Notes">
  - Ensure all event details are entered accurately before submission.
  - Public events will be visible to members, while private events are restricted to internal users.
  - The members can view their calendar of events in the monthly calendar.
</Note>
