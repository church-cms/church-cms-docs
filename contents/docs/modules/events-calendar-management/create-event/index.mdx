# How to Create an Event

## Step-by-Step Procedure

<Step>
  <StepItem title="Open the Calendar page">
    On the Calendar page, click the **Create Event** button located at the top-right corner.
  </StepItem>
  <StepItem title="Choose the event type">
    Choose the type of event from the dropdown:
    - **Private Event**
    - **Public Event**
  </StepItem>
</Step>

![Create Event Button](/modules/events-calendar/create-event.png)

    The event can be a private or a public event.
