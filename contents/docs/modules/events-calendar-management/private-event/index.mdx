# Creating a Private Event

<Step>
  <StepItem title="Select Private Event">
    Click **Create Private Event** from the dropdown menu.
  </StepItem>
  <StepItem title="Open Event Form">
    A new event form will open.
  </StepItem>
  <StepItem title="Enter the required details">
    Enter the required details:
    - **Title**
    - **Description**
    - **Repeats**
    - **Location**
    - **Event Category**
    - **Organized By**
    - **Start Date & Time**
    - **End Date & Time**
  </StepItem>
  <StepItem title="Submit the form">
    Click **Submit**.
  </StepItem>
</Step>

## Outcome

- The private event will be created successfully.
- It will be displayed in the calendar but limited to internal visibility.
