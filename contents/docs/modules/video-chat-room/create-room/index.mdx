## Step-by-Step Procedure

<Step>
  <StepItem title="Login to the system">
    Login to the system using **Admin credentials**.
  </StepItem>
  <StepItem title="Open Video Chat Room">
    From the dashboard, navigate to **Video Chat Room**.

    ![Video Chat Room Menu](/modules/video-chat-room/video-chat-room-menu.png)
  </StepItem>
  <StepItem title="Click the Add button">
    Click the **Add +** button located at the top-right corner of the page.
  </StepItem>
  <StepItem title="Enter the required details">
    A new form will open. Enter the required details:
    - **Title**
    - **Description**

    Then select the **Members** to be invited to the room.

    ![Create Room Form](/modules/video-chat-room/create-room-form.png)
  </StepItem>
  <StepItem title="Click Submit">
    Click **Submit** to create the room.
  </StepItem>
</Step>

## Outcome

- A new video chat room will be created successfully.
- Invited members can join the room for communication.

<Note type="info">
  Video communication is hosted on a secure cloud server, ensuring safe and reliable interaction.
</Note>
