# Edit Member Profile

## Steps

1. Open the required member profile.
2. Click the **Edit** option.

![Edit Staff Button](/modules/staff/staff-edit-button.png)

3. Make the necessary changes.

![Edit Staff Form](/modules/staff/staff-edit-form-1.png)

![Edit Staff Form](/modules/staff/staff-edit-form-2.png)

4. Click **Submit** to save the updated details.
