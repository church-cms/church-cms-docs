# How to Add a Sub Admin

## Step-by-Step Procedure

1. Login to the system using **Admin** credentials.
2. From the left navigation pane, click **Users**.
3. Select **Sub Admins**.

![Sub Admin List](/modules/sub-admin/sub-admin-list.png)

---

## Adding a New Sub Admin

1. Click the **Add** button.

![Add Sub Admin Button](/modules/sub-admin/add-sub-admin-button.png)

2. A new sub-admin registration form will open.

![Add Sub Admin Form](/modules/sub-admin/add-sub-admin-form-1.png)

![Add Sub Admin Form](/modules/sub-admin/add-sub-admin-form-2.png)

3. Enter all the required details.
4. Ensure the following key information is provided:
   - Email ID (Important for login access)
   - Other necessary profile details
5. Click **Submit**.

![Add Sub Admin Form](/modules/sub-admin/add-sub-admin-form-3.png)

![Add Sub Admin Form](/modules/sub-admin/add-sub-admin-form-4.png)

![Add Sub Admin Form](/modules/sub-admin/add-sub-admin-form-5.png)

---

## How to Use Search and Filter

- Records can be searched using the search bar by entering the name of the required entry.
- Users can also browse records using alphabetical sorting for quick shortlisting.
- This feature helps in easily finding specific members, parents, staff, or other records without manually scrolling through the entire list.

![Sub Admin Search](/modules/sub-admin/sub-admin-search-1.png)

![Sub Admin Search](/modules/sub-admin/sub-admin-search-2.png)

---

## Outcome

- The sub-admin profile will be created successfully.
- The new sub-admin will be listed in the **Sub Admins** section.

---

## Note

- The **Email ID** is mandatory as it is used for login and system communication.
