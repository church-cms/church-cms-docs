# Edit Member Profile

## Steps

1. Open the required member profile.
2. Click the **Edit** option.

![Edit Sub Admin Button](/modules/sub-admin/sub-admin-edit-button.png)

3. Make the necessary changes.

![Edit Sub Admin Form](/modules/sub-admin/sub-admin-edit-form-1.png)

![Edit Sub Admin Form](/modules/sub-admin/sub-admin-edit-form-2.png)

4. Click **Submit** to save the updated details.
