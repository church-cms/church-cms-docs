# How to Access Member Profile Actions

## Steps

1. Login to the system using **Admin** or **Staff** credentials.
2. Navigate to **Users → Sub Admins**.
3. Click the **name** and the profile will be opened.

![Sub Admin List Click](/modules/sub-admin/sub-admin-list-click.png)

---

## Sub Admin Profile Sections

The following tabs and details can be viewed in the profile:

- My Profile
- Timeline
- Family Tree
- Family
- Messages
- Notes
- Attendance

![Sub Admin Profile Overview](/modules/sub-admin/sub-admin-profile-overview.png)

---

## 1. Profile

Displays the complete profile information of the sub admin.

![Sub Admin My Profile Tab](/modules/sub-admin/sub-admin-my-profile-tab.png)

---

## 2. Timeline

Displays the activity history of the member profile. All updates and actions performed in the profile can be viewed here.

![Sub Admin Timeline](/modules/sub-admin/sub-admin-timeline-1.png)

![Sub Admin Timeline](/modules/sub-admin/sub-admin-timeline-2.png)

---

## 3. Family Tree

Displays the family details linked to the member. Family information such as:

- Name of the family member
- Relationship
- Date of Birth
- Email ID
- Other related details can be viewed here.

![Sub Admin Family Tree](/modules/sub-admin/sub-admin-family-tree-1.png)

![Sub Admin Family Tree](/modules/sub-admin/sub-admin-family-tree-2.png)

---

## 4. Family

Family tree details can be added while creating the member profile in the Users module.

![Sub Admin Family](/modules/sub-admin/sub-admin-family-1.png)

![Sub Admin Family](/modules/sub-admin/sub-admin-family-2.png)

---

## 5. Messages

Displays messages sent by the member. The following details can be viewed:

- Status
- Subject
- Message
- Attachments
- Sent On
- Action

![Sub Admin Messages](/modules/sub-admin/sub-admin-messages-1.png)

![Sub Admin Messages](/modules/sub-admin/sub-admin-messages-2.png)

---

## 6. Notes

Admin can add notes related to the member.

1. Enter the note in the provided field.
2. Click **Submit** to save the note.

![Sub Admin Notes](/modules/sub-admin/sub-admin-notes-1.png)

![Sub Admin Notes](/modules/sub-admin/sub-admin-notes-2.png)

---

## 7. Attendance

The attendance details of the member can be viewed in this section.

![Sub Admin Attendance](/modules/sub-admin/sub-admin-attendance-1.png)

![Sub Admin Attendance](/modules/sub-admin/sub-admin-attendance-2.png)

## Available Actions

Additional actions are available from the **⋮ menu** at the top-right of the profile. See [Menu Options](/docs/modules/user-management/sub-admin/menu-options) for the full list.
