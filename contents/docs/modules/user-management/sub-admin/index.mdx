# Sub Admin Management

## Introduction

The **Sub Admin Management** module allows administrators to create and manage sub-admin accounts with specific access privileges. This helps in delegating responsibilities while maintaining control over system operations.

---

## Accessing the Sub Admins List

Navigate to **Users → Sub Admins** from the main sidebar.

![Sub Admin List](/modules/sub-admin/sub-admin-list.png)

---

## Page Toolbar

The toolbar at the top of the Sub Admins list provides the following actions:

| Button | Description |
|---|---|
| **Add +** | Opens the [Add Sub Admin](/docs/modules/user-management/sub-admin/add) form |

---

## Sub Admins List Items

Each row in the list represents a sub-admin and shows key details at a glance. Click a sub-admin's **name** to open their [Sub Admin Detail](/docs/modules/user-management/sub-admin/detail) page.
