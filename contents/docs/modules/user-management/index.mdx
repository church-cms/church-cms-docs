# User Management

ChurchCMS organizes users into four distinct types. Each type carries a specific set of permissions, responsibilities, and access levels within the application.

## Role Comparison

| Capability | Member | Preacher | Sub-Admin | Staff |
|---|:---:|:---:|:---:|:---:|
| View own profile | ✓ | ✓ | ✓ | ✓ |
| Manage members | | | ✓ | ✓ |
| Manage preachers | | | ✓ | |
| Manage groups | | | ✓ | ✓ |
| Send messages | | | ✓ | ✓ |
| Access reports | | | ✓ | |
| Full admin access | | | | |

<Note type="info">
  The **primary Admin** account has full access across all modules and is the only role that can create or modify Sub-Admin accounts.
</Note>

## Navigation

All user type lists are accessible from the main sidebar under **Users**:

- **Users → Members**
- **Users → Preachers**
- **Users → Sub Admins**
- **Users → Staff**
