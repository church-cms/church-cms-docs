# Member Detail

The **Member Detail** page gives a complete view of a single member's record. It is divided into a **sidebar summary** on the left and a **tabbed detail panel** on the right.

## Opening a Member Profile

1. Go to **Users → Members**.
2. Click any member's **name** from the list.

---

## Sidebar Summary

The left sidebar shows the member's photo and core identity information at a glance.

### Actions

| Button | Description |
|---|---|
| **Edit** | Opens the [Edit Member](/docs/modules/user-management/members/edit) form |
| **Delete** | Permanently [removes the member](/docs/modules/user-management/members/delete) |

### Basic Information

| Field | Description |
|---|---|
| Family | Linked family group (if assigned) |
| Date of Birth | Member's date of birth |
| Occupation | Job title or role |
| Gender | Male / Female / Other |
| Aadhar Number | National ID (if provided) |

### Contact Information

| Field | Description |
|---|---|
| Address | Residential address |

---

## Profile Tabs

The right panel is organized into eight tabs. Click any tab heading to switch views.

### My Profile

Displays membership and personal details:

| Field | Description |
|---|---|
| Age | Calculated from date of birth |
| Blood Group | e.g., A, O+, B- |
| Marriage Status | Single / Married / Widowed |
| Marriage Date | Date of marriage (if applicable) |
| Membership Type | e.g., Member, Associate |
| Membership Start Date | When the membership was activated |

---

### Timeline

A chronological activity feed showing all updates, actions, and changes made to this member's record — including edits, group assignments, and message history.

---

### Family Tree

A visual representation of the member's family relationships in the system.

---

### Family

A tabular list of all linked family members.

| Column | Description |
|---|---|
| Name | Family member's full name |
| Relationship | e.g., Spouse, Child, Sibling, Parent |
| Date of Birth | For records and birthday tracking |
| Email | Contact email |
| Additional Info | Custom notes |

To add a family member, use [Menu Options → Add Family Member](/docs/modules/user-management/members/menu-options#add-family-member).

---

### Assigned Groups

Lists all groups this member belongs to (e.g., Youth, Choir, Prayer Team). Group assignments are managed from the **Groups** module.

---

### Messages

A log of all communications sent to this member.

| Column | Description |
|---|---|
| Status | Sent / Failed / Pending |
| Subject | Message subject line |
| Message | Body preview |
| Attachments | Linked files |
| Sent Date | Timestamp |

Click **View** on any row to read the full message thread.

---

### Notes

Admin-only private notes attached to the member record.

1. Type your note in the input field.
2. Click **Submit**.

<Note type="warning">
  Notes are visible only to **Admins**. They are not shared with the member.
</Note>

---

### Attendance

A log of all attendance events recorded for this member, linked from the **Attendances** module.

---

## Three-Dot Menu (⋮)

Additional actions are available from the **⋮ menu** at the top-right of the profile. See [Menu Options](/docs/modules/user-management/members/menu-options) for the full list.
