# Edit Member

The **Edit Member** form allows you to update any field on an existing member's profile.

## How to Open the Edit Form

From the [Member Detail](/docs/modules/user-management/members/detail) page:

1. Click the **Edit** button in the left sidebar.

The form opens pre-filled with the member's current information.

---

## Editable Fields

All fields from the original [Add Member](/docs/modules/user-management/members/add) form are editable, including:

**Personal Information**
- Full Name
- Date of Birth, Gender, Blood Group
- Aadhar Number, Occupation
- Marriage Status, Marriage Date
- Membership Type, Membership Start Date

**Contact Information**
- Email ID
- Phone Number
- Address

**Additional Information**
- Employment ID
- Family link
- Additional Notes

---

## Saving Changes

1. Make the desired changes.
2. Click **Submit**.

Changes take effect immediately and are reflected across the member's profile, membership card, and all linked records.

<Note type="info">
  Changing the **Email ID** will also update the member's login credentials. Notify the member if their email is changed.
</Note>
