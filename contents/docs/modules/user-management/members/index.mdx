# Members

The **Members** list is the main directory of all church members registered in ChurchCMS. It provides tools to browse, search, filter, and perform bulk actions on member records.

## Accessing the Members List

Navigate to **Users → Members** from the main sidebar.

The page header displays the total active member count (e.g., **Members (471)**).

---

## Page Toolbar

The toolbar at the top of the Members list provides the following actions:

| Button | Description |
|---|---|
| **Add +** | Opens the [Add Member](/docs/modules/user-management/members/add) form |
| **Export** | [Export](/docs/modules/user-management/members/export) the member list to a downloadable file |
| **Import** | [Import](/docs/modules/user-management/members/import) members in bulk from a file |
| **Generate Membership Card** | Bulk-generate printable membership cards for all members |

---

## Search

Use the **Search** bar to find members by name, email, phone number, or other profile fields. Results update as you type.

---

## A–Z Alphabetical Index

Below the toolbar, a row of letter buttons (`A` through `Z`) lets you filter the member list by the first letter of a member's name.

- Click any letter to show only members whose name starts with that letter.
- Click **CLEAR ALL** to remove the filter and show all members.

---

## Member List Items

Each row in the list represents a member and shows key details at a glance. Click a member's **name** to open their [Member Detail](/docs/modules/user-management/members/detail) page.
