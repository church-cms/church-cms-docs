# Add Member

The **Add Member** form lets you create a new church member record in ChurchCMS.

## How to Open the Form

1. Go to **Users → Members**.
2. Click the **Add +** button in the top-right corner of the toolbar.

---

## Form Fields

Fill in the following fields. Fields marked as **Required** must be completed before submission.

### Personal Information

| Field | Required | Description |
|---|:---:|---|
| Full Name | Yes | Member's first and last name |
| Date of Birth | Yes | Used for membership records and birthday tracking |
| Gender | Yes | Male / Female / Other |
| Blood Group | No | e.g., A+, O-, B+ |
| Aadhar Number | No | National ID number (if applicable) |
| Occupation | No | Member's job title or role |
| Marriage Status | No | Single / Married / Widowed |
| Marriage Date | No | Applicable if marriage status is Married |
| Membership Type | Yes | e.g., Member, Associate |
| Membership Start Date | Yes | Date the membership became active |

### Contact Information

| Field | Required | Description |
|---|:---:|---|
| Email ID | Yes | Used for login, verification, and communications |
| Phone Number | Yes | Primary contact number |
| Address | No | Residential address |

### Additional Information

| Field | Required | Description |
|---|:---:|---|
| Employment ID | No | Internal staff reference number |
| Family | No | Link to an existing family group |
| Additional Notes | No | Free-text field for any extra information |

---

## Submitting the Form

Once all required fields are filled:

1. Review all entries for accuracy.
2. Click **Submit**.

### Result

- The member is created and immediately appears in the [Members list](/docs/modules/user-management/members).
- A confirmation is shown at the top of the page.

---

## After Adding

Once created, you can:

- Open the member's [Detail page](/docs/modules/user-management/members/detail) to complete their profile.
- Add family members via the [Menu Options](/docs/modules/user-management/members/menu-options).
- Generate their [Membership Card](/docs/modules/user-management/members/menu-options#generate-membership-card).

<Note type="warning">
  The **Email ID** must be unique across all users. Duplicate emails will be rejected at submission.
</Note>
