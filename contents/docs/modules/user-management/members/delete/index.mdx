# Delete Member

Deleting a member permanently removes their record and all associated data from ChurchCMS.

## How to Delete a Member

From the [Member Detail](/docs/modules/user-management/members/detail) page:

1. Click the **Delete** button in the left sidebar.
2. A confirmation dialog will appear — review the member's name before proceeding.
3. Click **Confirm** to permanently delete.

---

## What Gets Deleted

When a member is deleted, the following data is permanently removed:

| Data | Removed |
|---|:---:|
| Member profile | ✓ |
| Messages & communication logs | ✓ |
| Notes | ✓ |
| Attendance records | ✓ |
| Family links | ✓ |
| Group assignments | ✓ |
| Membership card | ✓ |

<Note type="danger">
  Deletion is **irreversible**. There is no recovery option once confirmed. If you may need to retain the record, use **Deactivate** or **Exit Member** from the [Menu Options](/docs/modules/user-management/members/menu-options) instead.
</Note>

---

## Alternatives to Deletion

| Action | When to Use |
|---|---|
| [Deactivate Member](/docs/modules/user-management/members/menu-options#deactivate-member) | Member is temporarily inactive but may return |
| [Exit Member](/docs/modules/user-management/members/menu-options#exit-member) | Member has left the church permanently but records must be retained |
| **Delete** | Data is no longer needed and must be fully removed |
